Machine learning toolbox for Wavelet Packet Transform (WPT) and Ensemble Empirical Mode Decomposition (EEMD)¶

This toolbox includes the documentation for the Python codes that extract features by using Wavelet Packet Transform (WPT) and Ensemble Empirical Mode Decomposition (EEMD) and diagnose chatter in turning process. Algorithms are based on the methods explained in [Yesilli2019].
